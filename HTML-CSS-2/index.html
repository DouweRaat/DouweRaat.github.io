Projecten

Bij de Coderclass doen we soms ook projecten. Hier zie je al de projecten die ik heb gedaan.

Het eerste project dat ik heb gedaan is het Blender project. We moesten daarbij een video maken voor Warchild, dit is een organisatie die voor kinderen in oorlogsgebieden opkomt. Dit project heb ik samen met Leo Willems en Jacques Lamé. Dit is de link waar de video in staat. Door dit project heb ik ook de Blender-0 badge gekregen.

Het tweede project dat ik heb gedaan is het Philps Hue project. Hier moesten we een website maken die de Hue lampen aan en uit konden zetten. Ik zat in een groepje met Karsten Veltman, David Rebel en Divya Bhulai. We hadden het idee om een shipping bot te maken. Je kan dan twee namen invullen en dan laat hij zien hoe goed ze bij elkaar passen. De Hue lampen veranderden van kleur op basis hoe goed ze bij elkaar passen.

Bij het derde project dat ik heb gedaan moest je een sleutelhanger maken met Adobe Flash. Je kon je sleutelhanger laten uitprinten als een houten of een plastic sleutelhanger je kon er ook een sticker van maken, ik heb er een houten sleutelhanger van gemaakt.
